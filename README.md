# music-notes

Программный прототип для автоматической конвертации монофонической акустической музыкальной записи в нотное представление формата MusicXML.

Проект разработан в рамках выпускной квалификационной работы на тему «Разработка системы автоматической конвертации акустической музыкальной записи в нотную».

## Возможности

* загрузка аудиофайла в формате WAV;
* предварительная обработка аудиосигнала;
* определение основной частоты с помощью алгоритма pYIN;
* сегментация сигнала на нотные события и паузы;
* квантование длительностей;
* формирование нотной записи;
* экспорт результата в формат MusicXML.

## Структура проекта

```text
music-notes/
├── data/              # тестовые аудиозаписи
├── out_test/          # результаты транскрипции в формате MusicXML
├── src/               # исходный код программы
├── README.md          # описание проекта
└── requirements.txt   # список используемых библиотек
```

## Установка зависимостей

```bash
pip install -r requirements.txt
```

## Пример запуска

```bash
python src/transcribe.py data/test1.wav -o out_test/out_test1.musicxml
```

После выполнения программы в папке `out_test` формируется файл MusicXML, который можно открыть в нотном редакторе.

## Используемые библиотеки

* librosa;
* music21;
* numpy;
* scipy;
* matplotlib.

## Назначение проекта

Прототип предназначен для исследования основных этапов автоматической транскрипции монофонических музыкальных записей. Реализация позволяет проследить путь от обработки аудиосигнала до формирования итоговой нотной записи и проанализировать причины возможных ошибок транскрипции.
